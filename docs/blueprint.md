# **App Name**: ThesisEval

## Core Features:

- Real-Time Evaluation: Real-time evaluation interface for professors to assess student presentations, with fields for each evaluation criteria per semester.
- Dynamic Criteria Display: Displays semester-specific criteria for evaluation (Problem clarity, Methodological progress, Interim results, Final results, Extension justification).
- Responsive Design: A clean, responsive interface accessible on mobile, tablet, and desktop devices.
- Role-Based Authentication: Secure authentication system with distinct profiles for students, professors, and administrators.
- Record Identification: Record the evaluator (professor), the student being evaluated (name, ID), and the semester.
- AI Evaluation Assistant: A tool to receive natural language prompts (via text input), from professors, and generate helpful evaluation comments or insights.

## Style Guidelines:

- Primary color: Soft blue (#7BB4E5) to convey trust and reliability in the evaluation process.
- Background color: Light gray (#F0F4F8) to provide a neutral, distraction-free backdrop.
- Accent color: Muted green (#90EE90) to highlight key actions and provide positive feedback.
- Body and headline font: 'Inter', a sans-serif font, for clear readability across all devices.
- Use clean, professional icons to represent different evaluation aspects and system functions.
- Maintain a clean, intuitive layout with clear sections for evaluation input, real-time feedback, and report generation.
- Use subtle transitions and animations to enhance user experience during evaluation and data updates.
